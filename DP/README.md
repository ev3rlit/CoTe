### 학습 포인트

```
① 1D 기초 (1~5번)
   → "점화식을 세우고 반복문으로 채운다"의 감을 잡는 단계
   → dp[i]의 의미를 명확히 정의하는 연습

② 1D 응용 (6~10번)
   → "무엇을 인덱스로 잡는가"의 선택이 핵심
   → 순회 방향(정순/역순)에 따라 결과가 달라지는 것을 체감

③ 2D 기초 (11~12번)
   → State에 차원이 하나 추가되는 것에 익숙해지는 단계
   → dp[i][j]의 두 축이 각각 무엇을 의미하는지 정의
```
